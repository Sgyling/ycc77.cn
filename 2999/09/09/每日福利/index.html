2.乙巳年 戊寅月 辛亥日(25年2月11号) 福利-去除360所有广告

教程链接:https://pan.quark.cn/s/97fab39427ba

或者看下面的教程(因为网站的不稳定性,所以最好先保存,再观看)

官网下载:https://weishi.360.cn/?source=homepage/

极速版或者正常下载都可以哈

点击右上角的三条杠 -> 设置

基本功能 -> 功能定制

定制功能中-将以下内容关闭

取消选项:显示首页安全播报

取消选项:显示主界面搜索框

取消选项:显示360个人中心

取消选项:显示加速球

取消选项:显示办公工具(重启卫士后生效)

提醒设置中-关闭以下内容

取消选项:开启软件管家升级提示(发现软件管家有可升级的软件,将在主界面展示提醒标识)

系统右键菜单中-关闭以下内容

取消选项:把”使用360解除占用”添加到鼠标右键菜单

取消选项:把”使用360强力删除”添加到鼠标右键菜单

取消选项:把”使用360管理右键菜单”添加到鼠标右键菜单

取消选项:把”开启360桌面助手”添加到鼠标右键菜单

系统右键菜单中-保留以下内容

把”使用360进行木马云查杀”添加到鼠标右键菜单

把”深度清理垃圾”添加到鼠标右键菜单(仅右键单机”回收站”时有效)

升级设置中,选择:不自动生气

并取消”不升级卫士,但仍然升级备用木马库”

用户体验改善计划

取消选项:参加”360用户体验改善计划”

开启Defender

开启选项: 在系统操作中心不显示360防护信息,以便同时开启Windows Defender防护

注:如果你电脑上有多个软件杀毒,可以开启此选项,否则不要开启.

AI办公设置

取消选项:AI办公桌面图标创建

安全提示

防护弹窗模式,选择智能模式

聊天安全提示

取消选项:接收到文件为未知时提示

取消选项:接收到加密压缩包时提示

其他默认,不需要改动

下载安全提示

取消选项:下载文件为未知时提示

取消选项:下载加密压缩包时提示

取消选项:下载非office文件(.pdf等)时提示

其他默认,不需要改动

文件安全下载提示

取消选项:开启音视频安全下载提醒

取消选项:开启图片安全下载提醒

取消选项:开启apk文件安全下载提醒

用户账号及任务

取消选项:账号自动登录后提示

取消选项:账号等级升级时提示

取消选项:有新特权时提示

垃圾清理提示

取消选项:开启空闲提示

软件垃圾提醒设置

取消选项:浏览器退出时提醒

开启选项:自动清理

软件卸载残留提醒设置

取消选项:关闭软件卸载残留提醒

清理pro

取消选项:显示清理pro资源管理器入口

软件自启动提醒

选择选项:不提醒,自动优化

如果你是新手,请选择新增启动项时提醒(默认)

游戏优化结果提醒

取消选项:开启游戏优化结果提醒

开机小助手功能管理

取消选项:我愿意参加360开机时间调查

取消选项:开机后提示本次开机时间

取消选项:健康提醒

取消选项:显示天气预报等功能信息

取消选项:长时间未关机检测电脑性能

开机小助手广告管理 -> 不定期开机推广开关设置 -> 会出现’广告偏好设置’弹窗 -> 滚动到最下面 -> 取消选项:不定期开机推广开关设置

安全防护中心 -> 看片安全防护 -> 取消选项:开启360看片安全防护

搜索安全防护 -> 取消选项:使用搜索引擎防跟踪,保护隐私安全

核晶防护 -> 去管理 -> 开启核晶防护

U盘安全防护 -> 取消选项:自动扫描 -> 取消选项:插入U盘后显示U盘悬浮窗 -> 取消选项:插入U盘后再任务栏显示U盘小图标 -> 取消选项: 插入U盘后再系统窗口的侧边展现U盘内容

浏览器防护 -> 取消选项:当默认浏览器设置被修改时,弹窗提醒

开发着模式 -> 开启选项:将编译程序及编译输出的路径加入新人列表,减少弹窗提示,提高开发效率

漏洞修复方式 -> 选择选项:电脑关机时自动修复

木马查杀 -> 取消选项:开启内容推荐提醒 -> 取消选项:开启风险软件扫描提示

推广设置 -> 取消选项:开启产品推荐提醒 -> 取消选项:开启个性化内容推荐 -> 取消选项:开启个性化广告推荐 -> 取消选项:AI大会员提醒 -> 取消选项:开启提醒,可享受专属定制服务

广告咨询设置 -> 取消选项: 每日精选提醒中所有选项 -> 取消选项:热点资讯提醒中所有选项

点击确定以后 -> 回到主页面,再点击右上角的三条杠 -> 找到广告咨询设置 -> 广告退关管理 -> 广告开关设置 -> 滚动到最下面 -> 取消选项:广告展示设置 ->资讯推荐管理 -> 咨询开关设置 -> 滚动到最下面 -> 取消选项: 资讯展示设置 -> 确定

最后关闭360,重新启动360即可
